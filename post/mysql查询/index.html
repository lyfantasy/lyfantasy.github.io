select基本查询

Mar 18, 2023
1 min read

select语句

基本语句

select * from table_name;
  • 去重语句 distinct去掉重复的结果
select distinct user_role from user;
  • 别名as或者使用空格
select user_role as u_role from user;
select user_role u_role from user;

  • 空值参与运算 1.空值 NULL 2.空值不等于0，’’,’null’ 3.空值参与运算，结果也一定为空

  • 着重号 着重号用以区分关键字和字段。

select * from 'order';
  • 显示表结构
describe table_name;

where进行过滤

select * from user where list_name = 'value';

运算符

排序与分页

  • 排序顺序

    1.asc(升序排序) 2.desc(降序排序) 默认按照升序排序

select * from user order by user_salary desc;
  • 二级排序 先按照以及排序，当一级排序后存在相同的结果时，按照二级进行排序
select * from user order by user_salary desc , user_num asc;
  • 分页 使用limit关键字进行分页limit(偏移量,条目数)
# 查询20条数据，偏移量20，即查询20到40这一段数据
select * from user limit 20,20;

多表查询

  • 内连接 查询交集部分
select u.user_name,a.item_address
from user u inner join item a 
on u.user_item_id = a.item_id;
  • 左外连接 查询左边的全部和与右边的交集部分
select u.user_name,a.item_address
from user u left outer join item a 
on u.user_item_id = a.item_id;
  • 右外连接 查询右边全部和与左边的交集部分
select u.user_name,a.item_address
from user u right outer join item a 
on u.user_item_id = a.item_id;
  • 联合查询 将两次查询的结果拼成一个结果集
#全外查询
select table_list 
from table_a full outer join table_b 
on table_a.param = table_b.param;
  • 自然连接 查询两张表中的相同字段，然后自动连接，相当于等值连接
select table_list 
from table_a natural join table_b;
  • using关键字 自动查询using关键字后面两表之间相同的关键字，然后进行连接。
#查询两表中的user_item_id，然后进行连接
select table_list
from table_a join table_b
using(user_item_id);
